3) research about all git hub commands and understand the differences
https://www.git-tower.com/blog/git-cheat-sheet/

	add- adds all current changes to the next commit
	log- shows all commits starting with the newest
	merge- merges specified branch's history into the current one
	status- shows modified files in working directory, staged for next commit
	branch- list branches. 
	checkout- switch to another branch and check it out into working directory
	fetch- fetch down all the branches from that Git remote
	stash- saves modified and staged changes
